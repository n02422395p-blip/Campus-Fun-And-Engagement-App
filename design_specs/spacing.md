# Spacing System

## Base Unit: 8px

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon spacing, small gaps |
| sm | 8px | Between related items |
| md | 16px | Card padding, section gaps |
| lg | 24px | Major section spacing |
| xl | 32px | Screen margins |
| 2xl | 48px | Large sections |

## Screen Margins
- Left/Right margins: 20px
- Top padding (safe area): 44px (iOS) / 24px (Android)
- Bottom padding: 34px (iOS) / 24px (Android)

## Component Spacing
- **Cards**: Padding 16px, margin bottom 16px
- **Buttons**: Padding vertical 12px, horizontal 24px
- **Form fields**: Margin bottom 20px
- **Lists**: Gap between items 8px

## Touch Targets (Minimum)
- Buttons: 44px height
- Icons: 44x44 hit area
- List items: 50px height
